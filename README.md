# 🤖 AI Job Hunter Agent

An AI-powered automation workflow built with n8n that helps automate the job application process.

---

## Features

✅ Scrapes LinkedIn jobs using Apify

✅ Filters jobs using Gemini AI

✅ Generates a tailored resume for each relevant job

✅ Creates Google Docs automatically

✅ Saves application details to Google Sheets

---

## Tech Stack

- n8n
- Google Gemini
- Apify
- Google Docs API
- Google Sheets API

---

## Workflow

1. Search LinkedIn jobs
2. Extract job descriptions
3. Customize resume
4. Save application to Google Sheets
